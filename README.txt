The project aims towards creating a hierarchy of banking system. In this project, we have built Bank System in Java. This project gives an overview of functionality of Java. In this project, the users will be able to perform the following functionalities:
1.	Account Management
2.	Loan Management 
3.	Insurance Management

It takes in current date and time and gives notification of loans, insurances and calculates intrest on accounts. It also has the functionality of creating an account, draw a loan, or to take an insurance. There is a userid-password system for different users that the programs keeps track of.
